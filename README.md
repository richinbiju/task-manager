# Task Manager

Modern microfrontend + microservices architecture task manager.

## Tech Stack

- React
- Vite
- Tailwind v4
- Module Federation
- Node.js
- Express
- PostgreSQL

## Architecture

Frontend:
- Shell App
- Auth Microfrontend
- Tasks Microfrontend
- Dashboard Microfrontend

Backend:
- API Gateway
- Auth Service
- Task Service
